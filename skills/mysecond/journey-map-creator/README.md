# Journey Map Creator

Build detailed customer journey maps that identify pain points and opportunities across touchpoints.

---

## Installation (30 seconds)

**In Claude Code Desktop:**
1. Unzip this file
2. Move the `journey-map-creator` folder to `~/.claude/skills/` (works across all projects)
3. Restart Claude Code Desktop

**Advanced:** For project-specific use, place in `.claude/skills/` within your project folder instead.

**In Claude Code Terminal:**
1. Unzip this file
2. Move the `journey-map-creator` folder to `~/.claude/skills/`
3. Restart Claude Code

**In Claude.ai (Chat/Desktop):**
1. Download this ZIP file (don't unzip)
2. Go to **Settings → Capabilities → Skills**
3. Click **+ Add** → **Upload a skill**
4. Select the ZIP file
